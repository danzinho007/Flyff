2- Como Eo sistema de classes no jogo ?
Lv 15 vocEobtém a 1° classe 
Lv 60 = 2° Classe
Lv 120 = Classe Master
Lv 120-M = Classe Hero
Lv 130-H = 3° Classe
Lv 150-L = Classe Lenda

1- Quais são os tipos de elementos no jogo ?
Generator = Elétrico
Flame = Fogo
River = Água
Cyclone = Vento
Desert = Terra

2- Qual elemento ganha de qual ?

Elétrico ganha de Água > Fogo > Vento > Terra > Elétrico

3- Quais são os atributos ?

STA = Vigor
DEX = Destreza
STR = Força
INT = Inteligência

4- Quais atributos das classes ?

Assist :
STR = 1x1 BP
INT = AOE BP ou Ringmaster
STA = AOE BP
DEX = Taxa de acerto

Mágicos :
STA = Aumenta a vida , reduz dano recebido
INT = Aumenta o dano por ataque mágico e aumenta a mana

Acrobat ( Arqueiro ) :
Arco : DEX ou STA
Yoyo : STR ou STA
